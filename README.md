# MAT
MAT: Multihop Annotation Tool

## Installation

```sh
conda create -n multihop-annotation-tool python=3.10

conda activate multihop-annotation-tool

pip install -r requirements.txt
```

## Usage

```sh
./run.sh
```

Access via http://127.0.0.1:8000
